1. Quick start
1.1. Follow [Quick Start][quick-start] of Jekyll to create a blog template.

1.2. Commit your site to github
Github reserve repository name `username.github.io` for user to publish a static website.
1.2.1 Create a new repository with name as `username.github.io`, for example my website repository is [vewe-richard.github.io][vewe-richard-repository].
1.2.2 Add all files of your site to this repository.
1.2.3 Preview your blog by --- `http://username.github.io`

[quick-start]: https://jekyllrb.com/docs/
[vewe-richard-repository]:https://github.com/vewe-richard/vewe-richard.github.io

to maintain your blog, you need learning more about Jekyll, markdown
